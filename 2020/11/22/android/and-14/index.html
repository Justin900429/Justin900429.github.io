前言

  2. 建立 bottom navigation bar

Bottom navigation bar

Menu 製作

bottom navigation bar 指的是就是在畫面底下的 bar （這不是廢話嗎XD）。如下圖：



因此我們先來製作 navigation bar 吧！首先我們 navigation bar 既然是 bar，內部的 items 依然是由 menu 來控制的，因此我們必須要創建 menu，詳細步驟請參考這裡。之後創建一個 nav_bar.xml。

名字可以依照各自喜好，但是要易於維護。

另外我們可以利用 Android 的 vector asset 來製作 icon。我們在 drawable 的資料夾點擊右鍵


跳出的頁面中，我們可以在 Name 輸入我們想創建的 icon 名字。Clip Art 可以搜尋想要的 icon，但不是所有的 icon 都找得到。另外也提供 Color、Size 等屬性做調整。



創建後就會在 drawable 裡發現新增的 xml 檔案。


直接放上我 nar_bar.xml 的程式碼

  • nav_bar.xml
    <?xml version="1.0" encoding="utf-8"?>
<menu xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto">
    <item
        android:id="@+id/fragment_1"
        android:icon="@drawable/android_for_fragment"
        android:title="fragment 1"/>

    <item
        android:id="@+id/fragment_2"
        android:icon="@drawable/calendar_for_fragment"
        android:title="fragment 2" />

    <item
        android:id="@+id/change"
        android:icon="@drawable/back_for_fragment"
        android:title="change" />
</menu>

其中我們每個 item 沒有加上 showAsAction，預設為 ifRoom。

即使設成 never 也不會收藏起來。

做完後會出現我們的 menu。


製作 Bottom navigation bar

首先我們要先在 gradle 的 dependencies 裡面加入：

implementation 'com.google.android.material:material:1.2.0'

因為我們會用到 material 裡面的 widget。

接著開啟我們的 activity_main.xml，並在裡面加入：

<com.google.android.material.bottomnavigation.BottomNavigationView
    android:id="@+id/bottom_navigation"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:background="?android:attr/windowBackground"
    app:itemBackground="@color/colorPrimary"
    app:itemIconTint="#fff"
    app:itemTextColor="#fff"
    app:menu="@menu/nav_bar" 
    app:layout_constraintBottom_toBottomOf="parent"
    app:layout_constraintEnd_toEndOf="parent"
    app:layout_constraintStart_toStartOf="parent"/>

這邊我說明一些比較特別的來說明

  • android:itemBackground
    設定背景顏色
  • app:itemIconTint
    icon 的顏色
  • app:itemTextColor
    字體的顏色
  • app:menu
    選擇我們的 bottom navigation bar 要用哪個 menu。

此時就會看到我們成功製作 bottom navigation bar。



Listener

我們可以利用 Listener 來監控使用者按了哪個 option。這邊使用的 listener 是setOnNavigationItemSelectedListener。

// Set item listener of the bottom navigation bar
bottom_navigation.setOnNavigationItemSelectedListener {
    // Use the itemId to detect which button been pressed
    when (it.itemId) {
        ...
    }

    true // Return true
}

FrameLayout

FrameLayout 為 layout 的其中一種佈局樣式。當我們使用 FrameLayout，程式會為我們做留白，讓我們可以自由加入我們想要的內容。這裡就是為了放入我們的 Fragment，我們可以自由切換 FrameLayout 裡的 layout 是 FragmentOne 還是 FragmentTwo。

這邊我們將他加入到我們的 activity_main.xml 裡。

<FrameLayout
    android:id="@+id/fragment_holder"
    android:layout_width="match_parent"
    android:layout_height="0dp"
    android:layout_marginBottom="50dp"
    app:layout_constraintBottom_toTopOf="@+id/bottom_navigation"
    app:layout_constraintEnd_toEndOf="parent"
    app:layout_constraintHorizontal_bias="1.0"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintTop_toTopOf="parent"
    app:layout_constraintVertical_bias="0.50" />

supportFragmentManager

我們可以利用 FragmentManager 來幫助我們管理我們 Fragment 的狀態以及生命週期。我們可以在我們的 class 裡宣吿：

private val manager = supportFragmentManager

建立我們的 FragmentManager 之後，我們就可以用 manager 來新增、刪除或取代我們的 Fragment。這邊我們是要在 FragmentOne 跟 FragmentTwo 中切換，因此我們可以利用取代。

我們要利用到 FragmentManager 裡的 transaction，因此我們可以宣吿:

val transaction = manager.beginTransaction()

之後我們如果想將 FrameLayout 取代成 FragmentOne，我們可以：

// Get the fragmentOne class
val fragment = FragmentOne()

// Replace Framelayout with fragment
transaction.replace(R.id.fragment_holder, fragment)

// Commit the transaction
transaction.commit()

因為我們有 FragmentOne 跟 FragmentTwo，因此我們利用 showFragment1 跟 showFragment2 來包裝我們的程式。其中我們利用 isFragmentOne 來判斷現在 FrameLayout 是哪一個 Fragment。

private fun showFragment1() {
    // Create transaction to transact the FrameLayout
    val transaction = manager.beginTransaction()
    // Create FragmentOne class
    val fragment = FragmentOne()

    // Replace FrameLayout with fragment
    transaction.replace(R.id.fragment_holder, fragment)
    // Commit the transaction
    transaction.commit()

    // Set isFragmentOne to true
    isFragmentOne = true
}

private fun showFragment2() {
    // Create transaction to transact the FrameLayout
    val transaction = manager.beginTransaction()
    // Create FragmentTwo class
    val fragment = FragmentTwo()

    // Replace FrameLayout with fragment
    transaction.replace(R.id.fragment_holder, fragment)
    // Commit the transaction
    transaction.commit()

    // Set isFragment to False
    isFragmentOne = false
}

  • MainActivity.kt

class MainActivity : AppCompatActivity() {
    // Boolean to detect whether the layout is FragmentOne or Two
    private var isFragmentOne = true;
    // Fragment Manager to manage the layout of the fragment
    private val manager = supportFragmentManager

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        // Initial set the FrameLayout to FragmentOne
        showFragment1()

        // Set item listener of the bottom navigation bar
        bottom_navigation.setOnNavigationItemSelectedListener {
            // Use the itemId to detect which button been pressed
            when (it.itemId) {
                R.id.fragment_1 -> showFragment1()
                R.id.fragment_2 -> showFragment2()
                else -> {
                    if (isFragmentOne) {
                        showFragment2()
                    } else {
                        showFragment1()
                    }
                }
            }

            true
        }
    }

    private fun showFragment1() {
        // Create transaction to transact the FrameLayout
        val transaction = manager.beginTransaction()
        // Create FragmentOne class
        val fragment = FragmentOne()

        // Replace FrameLayout with fragment
        transaction.replace(R.id.fragment_holder, fragment)
        // Commit the transaction
        transaction.commit()

        // Set isFragmentOne to true
        isFragmentOne = true
    }

    private fun showFragment2() {
        // Create transaction to transact the FrameLayout
        val transaction = manager.beginTransaction()
        // Create FragmentTwo class
        val fragment = FragmentTwo()

        // Replace FrameLayout with fragment
        transaction.replace(R.id.fragment_holder, fragment)
        // Commit the transaction
        transaction.commit()

        // Set isFragment to False
        isFragmentOne = false
    }
}
