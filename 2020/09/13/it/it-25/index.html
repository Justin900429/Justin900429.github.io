前言

今天要介紹的演算法屬於 Unsupervised。這個演算法個概念非常簡單，今天我們也會自己實作這個演算法！

K-means

K-means 的直覺概念就是誰離我近你就是我們的類別！因此在整個演算法最初的概念就是先找幾個資料點當作我們的標準點，每個標準點代表一個種類的中心，再算各點到這些標準點的距離，離哪個標準點愈近，就是那個種類的點。而新的標準點會是在分配完各點後，取各種類的平均當作新的標準點。


從上面這張圖我們可以看到最初我們也是從所有的資料點挑 $n$ 個點當我們的標準點（$n$ 代表我們的類別）。接著算每個點到各 標準點 的距離，在取新的 類別的平均，重複這些動作直到收斂。

Distance

在介紹看程式碼之前，我們會先介紹不同的距離算法，讓大家對 K-means 更有感覺！

Euclidean distance

我相信歐式距離大家肯定都是耳熟能詳，但是我們這裡還是說明一下:

假設我們有兩點$A(a_1, b_1), B(a_2, b_2)$，其距離會是兩點的最短直線（上圖綠色），長度為：

$$
\overline{AB} = \sqrt{(a_1 - a_2)^2 + (b_1 - b_2)^2}
$$

Manhattan distance

曼哈頓距離大家可能比較少用過，同樣是$A, B$ 兩點，Manhattan distance 指的是紅色虛線加起來的距離。也就是:

$$
\overline{AB} = \mid a_1 - a_2 \mid + \mid b_1 - b_2 \ \mid
$$

這個距離可能會用在地圖上，因為我任兩個點不太可能直接穿越，可能會有街區啊，建築等阻擋，因此距離就會用 Manhattan distance。

程式碼

這邊我只貼實作的部分，利用 sklearn 的方式一樣在 Notebook。

在算各點的距離公式時，我這邊是直接用 sklearn 裡的函式 pairwise_distances_argmin。這個函數吃我們的資料以及中心點。之後會回傳一個矩陣紀錄各點距離哪個中心點最近。

def find_clusters(X, n_clusters):
    # Randomly choose clusters from origin datasets
    point_index = np.random.randint(low=0, high=X.shape[0] - 1, size=n_clusters)
    centers = X[point_index]
    
    while True:
        # Assign labels based on closest center
        labels = pairwise_distances_argmin(X, centers)
        
        # Find new centers from means of points
        new_centers = np.array([X[labels == i].mean(axis=0)
                                for i in range(n_clusters)])
        
        # Check for convergence
        if np.all(centers == new_centers):
            break
        centers = new_centers
    
    return centers, labels
