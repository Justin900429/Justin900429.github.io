前言

Toast

重溫

之前我們有提過 Toast 的使用方法會在底下提示，用法為：

Toast.makeText(this, <message>, <time>).show()

自訂 Toast

其實我們也可以將我們的 Toast 視為一個物件，並改變他的 properties.

// Make a Toast() object
val toast = Toast(this)

// Properties of toast
toast.setGravity(Gravity.TOP, 10, 10) // Set position
toast.duration = Toast.LENGHT_SHORT // time for toast to show
toast.view = layoutInflater.inflate(R.layout.toast_layout, null) // Custom layout

// Show the toast
toast.show()

我們來看一下我們在這邊設定的 properties：

  • toast.setGravity
    第一個參數設定我們的位置 Gravity.TOP （上）、Gravity.RIGHT（右）
    、Gravity.LEFT （左）、Gravity.BOTTOM （下）、Gravity.CENTER （中間）。
    第二跟第三設定我們 x 跟 y 的偏移值。

  • toast.durate
    設定持續時間。有 Toast.LENGTH_LONG 跟 Toast.LENGTH_SHORT，分別是 2 秒跟 3.5 秒。

  • toast.view
    toast.view 可以設定使用者要求的 layout。使用方式跟我們之前說過的 inflate 的方式。後面 layout 的名字則是使用 **R.layout.**。

此外，在 toast.setGravity 可以透過 or 來組合顯示。例如說要在上方中央的話就是：

toast.setGravity(Gravity.TOP or Gravity.CENTER, 0, 0)

Alert Dialog

有時候我們在使用 app 的時，會有跳出視窗來體醒我們訊息，或是要求權限等等，這就是 alert dialog 的作用。例如底下的提示訊息。



Alert builder

要創立我們的 alert dialog 需要使用到 AlertDialog builder。同時我們也需要傳入要在哪個 Acitivity 出現。

val dialog = AlertDialog.builder(this)

標題需要用到 setTitle() 來設定。內容則是用 setMessage()。

dialog.setTitle("Pratice alert dialog")
dialog.setMessage("Hello World!")

按鈕類型

一般我們在做 Alert Dialog 時會提供三種按鈕給使用者，像上圖有 cancel、no、yes 等等。一般這三種對應到 setNeuralButton(), setPositiveButton(), setNegativeButton()。提供這三種按鈕可以為使用者的選擇而作出相對應的反應。



// Neutral button
dialog.setNeutralButton("Cancel") {dialog, which ->
    Toast.makeText(this, "Press 'cancel' button", Toast.LENGTH_SHORT).show()
}

// Positive button
dialog.setPositiveButton("Yes") {dialog, which ->
    Toast.makeText(this, "Press 'yes' button", Toast.LENGTH_SHORT).show()
}

// Negative button
dialog.setNegativeButton("No") {dialog, which ->
    Toast.makeText(this, "Press 'no' button", Toast.LENGTH_SHORT).show()
}

一般來說我們按鈕有兩個參數我這邊把它命名成 dialog 跟 which。雖然這邊用不到，但是我們還是要設定，讓 lambda 可以接受參數。箭頭後面就接要的功能。

最後要顯示則是要利用 show。

dialog.show()

Anonymous builder

上述的方式我們都需要建立物件在啟用功能，我們也可以簡化他。首先我們不需要指派物件，我們可以直接使用 AlertDialog builder，並對所有的 properties 都用 . 來呼叫。

AlertDialog.Builder(this)
    .setTitle("Practice alert dialog")
    .setMessage("Hello world!")
    .setNeutralButton("Cancel") {dialog, which ->
        Toast.makeText(this, "Press 'cancel' button", Toast.LENGTH_SHORT).show()
    }
    .setNegativeButton("No") {dialog, which ->
        Toast.makeText(this, "Press 'no' button", Toast.LENGTH_SHORT).show()
    }
    .setPositiveButton("Yes") {dialog, which ->
        Toast.makeText(this, "Press 'yes' button", Toast.LENGTH_SHORT).show()
    }
    .show()

Items

除了提示訊息，我們也可以讓使用著選擇選項。大致分為兩方式，setItems() , setSingleItems()。


前者方式會是屬於按下後就是最後選擇，後者則是可以讓使用者按 DONE 之後退出。

  • setItems()
    在參數方面我們需要傳入一個 string array。此物件為 java 中的 array，因此我們 不能使用 listOf，要使用 array<string> 。 

    val items = arrayOf<String>("1", "2", "3", "4", "5")

dialog.setItems(items) {dialogInterface, i ->
Toast.makeText(
    this, 
    "You choose ${items[i]}!",
    Toast.LENGTH_SHORT).show()
}

  • setSingleChoiceItems()
    在參數方面，我們除了要傳入 array，我們也需要給定預設值，例如 0 就是預設第一個物件被選擇。令外我們也可以設定一個變數 position 來記錄使用者選擇哪個物件。因此當使用者按下 DONE 之後，我們可以設定某個 TextView 為使用者選的結果。 

    val items = arrayOf<String>("1", "2", "3", "4", "5")
val position = 0;

dialog.setSingleChoiceItems(items, 0) {dialogInterface, i ->
    position = i;
}
dialog.setPositiveButton("Done") {dialog, which ->
    Toast.makeText(
        this, 
        "You choose ${position}!", 
        Toast.LENGTH_SHORT).show()
    txt_num.text = position.toString()
}

當然上述的方式也可以用 Anoymous 的方式來建立。

自訂 Toast

/*
Show custom toast message
 */
private fun showToast() {
    // Make Toast object
    val toast = Toast(this)

    // Set position
    toast.setGravity(Gravity.CENTER or Gravity.TOP, 0, 200)
    // Duration time
    toast.duration = Toast.LENGTH_SHORT
    // Custom view
    toast.view = layoutInflater.inflate(R.layout.toast_layout, null)

    // Show custom toast
    toast.show()
}

Alert Dialog with single choice item

完整 MainActivity 程式碼

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)

        // Button to show custom toast
        btn_custoa.setOnClickListener {
            showToast()
        }

        // Button to show Original alert dialog
        btn_orial.setOnClickListener {
            showOriginalAlert()
        }

        // Button to show items alert dialog
        btn_choal_1.setOnClickListener {
            showChoiceAlert1()
        }

        // Button to show items alert dialog
        btn_choal_2.setOnClickListener {
            showChoiceAlert2()
        }
    }

    /*
    Alert Dialog with multiple itmes
     */
    private fun showChoiceAlert1() {
        // Array for option
        val items = arrayOf<String>("1", "2", "3", "4", "5")

        // Alert Dialog
        AlertDialog.Builder(this)
            .setTitle("Choose a number!")
            .setItems(items) {dialogInterface, i ->
                Toast.makeText(this, "You choose ${items[i]}!", Toast.LENGTH_SHORT).show()
            }
            .show()
    }

    /*
    Alert Dialog with single choice item
     */
    private fun showChoiceAlert2() {
        // Array for option
        val items = arrayOf<String>("1", "2", "3", "4", "5")
        // Record the position
        var position = 0;

        // Alert Dialog
        AlertDialog.Builder(this)
            .setTitle("Choose a number!")
            .setSingleChoiceItems(items, 0) {dialogInterface, i ->
                position = i;
            }
            .setPositiveButton("Done") {dialog, which ->
                Toast.makeText(this, "You choose ${position}!", Toast.LENGTH_SHORT).show()
                txt_num.text = position.toString()
            }
            .show()
    }

    /*
    Alert Dialog with no choice items
     */
    private fun showOriginalAlert() {
        AlertDialog.Builder(this)
            .setTitle("Practice alert dialog")
            .setMessage("Hello world!")
            .setNeutralButton("Cancel") {dialog, which ->
                Toast.makeText(this, "Press 'cancel' button", Toast.LENGTH_SHORT).show()
            }
            .setNegativeButton("No") {dialog, which ->
                Toast.makeText(this, "Press 'no' button", Toast.LENGTH_SHORT).show()
            }
            .setPositiveButton("Yes") {dialog, which ->
                Toast.makeText(this, "Press 'yes' button", Toast.LENGTH_SHORT).show()
            }
            .show()
    }


    /*
    Show custom toast message
     */
    private fun showToast() {
        // Make Toast object
        val toast = Toast(this)

        // Set position
        toast.setGravity(Gravity.CENTER or Gravity.TOP, 0, 200)

        toast.duration = Toast.LENGTH_SHORT
        // Custom view
        toast.view = layoutInflater.inflate(R.layout.toast_layout, null)

        // Show custom toast
        toast.show()
    }
}
