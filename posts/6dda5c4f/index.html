前言

今天我們會說明如何在 App 裡創建 Menu

這篇看完後會了解到：
  1. 新增 Menu 以及顯示方式。
  2. 新增 Menu 的 Action。

Menu

一般我們在使用 App 時，右上方通常會出現選單來使用。一般我們可以將選單的內容放出來，也可以收納起來。

製作 Menu

在製作 Menu 之前，我們必須為其製作一個 resource folder 方便我們之後使用與分類。製作的方式也很簡單，在我們的左邊選單欄裡 res 按右鍵，按 New 並新增 Android Resource Directory。



之後會發現在 res 的目錄下多了 menu 這個資料夾。



此時就可以在 menu 底下新增 xml 的檔案。

撰寫 Menu

當我們創立好我們的 xml 的時候，我們會發現他的初始檔案是：

<?xml version="1.0" encoding="utf-8"?>
<menu xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto">

</menu>

這樣代表我們的 xml 初步已經創立好了。接著就要來新增我們的 menu 的選項了。

想要建立 option，我們就要用<item /> 的 tag 來創立。例如我想創建一個 contact 的 option，我就可以用

<item
    android:id="@+id/menu_contact"
    android:title="Contact"
    app:showAsAction="ifRoom"/>

其中 title 就是我們要顯示的文字，而 showAsAction 就是我們 option 顯示的方式。另外，除了顯示文字我們也可以利用 icon 的方式來製作圖表。我們只要將我們做好的 icon 放到 resource 的資料夾，或是利用 android 內建的 vector asset 來製作也可以。使用 icon 的方式就是加上 andorid:icon=，例如：

<item
    android:id="@+id/menu_heart"
    android:title="Heart"
    android:icon="@drawable/ic_baseline_favorite_24"
    app:showAsAction="ifRoom"/>

其中

android:icon="@drawable/ic_baseline_favorite_24"

是指取用 res/drawable 裡的 ic_baseline_favorite_24 至個 icon。

showAsAction

我剛剛在上面有提到說 showAsAction 可以用來控制我們 option 顯示方式，一般來說共有五種方式，分別是 ifRoom、 never、 always、 withText 跟 collapseActionView。

  • ifRoom
    如果在 Action Bar 還有空間的話會直接顯示，如果沒有空間就會移入三點選單起來。

  • never
    如果不想直接顯示，而是透過收納的方式，則可以使用這個。但是一定要給定 item 而不能單純用 icon。例如：

    <item
    android:id="@+id/menu_easter_egg"
    android:title="You found me!"
    app:showAsAction="never"/>


  • always
    不管 Action Bar 還有沒有空間，一律顯示出來不折疊。

  • withText
    盡可能將 title 跟 icon 一起顯示出來。會被放在 摺疊區。

  • collapseActionView
    可以將 ActionBar 的其他 icon 折疊起來。可以參考官方文檔。



Infate menu

雖然我們已經設計好介面，我們還是需要在 Activity 裡面利用 onCreateOptionsMenu 來製作我們的 menu。

/*
 Override the function to create menu
 */
override fun onCreateOptionsMenu(menu: Menu?): Boolean {
    // Inflate the menu xml
    menuInflater.inflate(R.menu.menu, menu)
    return true
}

這裡我們用之前講過的 inflate 來製作我們的 menu。其中 R.menu.menu 指的就是在 res/menu/menu.xml，如果是要 inflate test.xml 可以用 R.menu.test。

設定 Action

製作玩 menu 後，我們要來設定我們每個 option 的反應。這邊我們需要去 override onOptionsItemSelected。另外要注意的是，onOptionsItemSelected 不會只設定 menu 的 action，因此我們在設定 ID 時要設的不一樣才不會搞混。

/*
 Override the function to set up the action
 */
override fun onOptionsItemSelected(item: MenuItem): Boolean {
    // Set up the action for each items
    when (item.itemId) {
        R.id.menu_contact -> {
            Toast.makeText(this, "Don't have any contact", Toast.LENGTH_SHORT).show()
        }
        R.id.menu_heart -> {
            Toast.makeText(this, "To my belove", Toast.LENGTH_SHORT).show()
        }
        R.id.menu_easter_egg -> {
            Toast.makeText(this, "You found the easter egg!", Toast.LENGTH_SHORT).show()
        }
    }

    return super.onOptionsItemSelected(item)
}

這邊我們只是單純設定 Toast 來顯示按的是哪個 option，如果需要也可以做出換頁。

程式碼

  • menu.xml

    <?xml version="1.0" encoding="utf-8"?>
<menu 
    xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto">

    <item
        android:id="@+id/menu_contact"
        android:title="Contact"
        app:showAsAction="ifRoom"/>

    <item
        android:id="@+id/menu_heart"
        android:icon="@drawable/ic_baseline_favorite_black"
        android:title="Heart"
        app:showAsAction="collapseActionView"/>

    <item
        android:id="@+id/menu_easter_egg"
        android:title="You found me!"
        app:showAsAction="never"/>

</menu>

  • MainActivity

    class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }

    /*
     Override the function to create menu
     */
    override fun onCreateOptionsMenu(menu: Menu?): Boolean {
        // Inflate the menu xml
        menuInflater.inflate(R.menu.menu, menu)
        return true
    }

    /*
     Override the function to set up the action
     */
    override fun onOptionsItemSelected(item: MenuItem): Boolean {
        // Set up the action for each items
        when (item.itemId) {
            R.id.menu_contact -> {
                Toast.makeText(this, 
                               "Don't have any contact",
                               Toast.LENGTH_SHORT
                               ).show()
            }
            R.id.menu_heart -> {
                Toast.makeText(this,
                               "To my belove",               
                               Toast.LENGTH_SHORT
                               ).show()
            }
            R.id.menu_easter_egg -> {
                Toast.makeText(this,
                               "You found the easter egg!",
                               Toast.LENGTH_SHORT
                               ).show()
            }
        }

        return super.onOptionsItemSelected(item)
    }
}
