Views and ViewGroups

Views

Views 指的就是 UI 的元件，包括 TextView, Button, ImageView, EditText, CheckBox 等等零件。


ViewGroups

ViewGroups 指的就是 Views 的集合，一般常用的 ViewGroups 有三種排版的方式：

  1. ConstraintLayout
  2. LinearLayout
  3. RelativeLayout

零件間的關係

在設計我們的 layout 時我們會用到很多不同的 ViewGroups 或是 Views。而 Views 是只能有 parent 但不能有 child。另外 ViewGroups 的 child 可以為很多的 Views 和其他的 ViewGroups。

ConstrainLayout

ConstrainLayout 會要求物件被限制。比起其他的排版方式，利用 ConstrainLayout 可以用拖拉的方式來排版，但是要注意的是排版會根據不同手機而有不同效果，因此在用拖拉設計的時候要用比例而不是距離。注意在 xml 裡最外層是androidx.constraintlayout.widget.ConstraintLayout。

<?xml version="1.0" encoding="utf-8"?>
<androidx.constraintlayout.widget.ConstraintLayout
    xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:app="http://schemas.android.com/apk/res-auto"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="horizontal"
    tools:context=".MainActivity">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello World!"
        android:textSize="30sp"
        app:layout_constraintBottom_toBottomOf="parent"
        app:layout_constraintEnd_toEndOf="parent"
        app:layout_constraintStart_toStartOf="parent"
        app:layout_constraintTop_toTopOf="parent" />
        
</androidx.constraintlayout.widget.ConstraintLayout>

我們可以注意到我們每個物件都會多出四個點，我們需要拖拉到其他物體來限制我們這個物體。一般來說我們只要在垂直跟水平方向各限制至少一個就好。

layout_width and layout_height

我們在以後的物件都一定要設定這兩個值，這兩個值代表我們物件的外圍有多大。一般來說我們有:

  • wrap_content
    長度視物件的大小決定。
  • match_parent
    跟 parent 一樣大。這邊我們設定 match_parent。即便拉到四邊，因為我們的外圍大小跟 parent 一樣，因此我們不會維持在中間。

LinearLayout

LinearLayout 簡單說就是利用連續排版的方式，因此有兩種的排版的方向：

  • horizontal
  • vertical

我們先來看看 vertical 的效果：

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
    xmlns:tools="http://schemas.android.com/tools"
    android:layout_width="match_parent"
    android:layout_height="match_parent"
    android:orientation="vertical"
    tools:context=".MainActivity">

    <TextView
        android:id="@+id/textView"
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello World!"
        android:textSize="30sp" />

    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="Hello World!"
        android:textSize="30sp" />

</LinearLayout>

如果將上面 LinearLayout 的 orientation 換成 horizontal 會變成右邊的圖像。
